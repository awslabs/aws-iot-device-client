Force all consumers to stop waiting so that they can flush the queue and end any waiting behavior that might prevent the thread from shutting down.

This function essentially shuts off any of the 'waiting' behavior when it comes to getting the next message in the LogQueue. It will force the getNextLog() method to return whether there is a log message or not
