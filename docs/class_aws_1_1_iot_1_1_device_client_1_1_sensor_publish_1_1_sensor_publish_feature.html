Detailed Description

SensorPublish publishes sensor data captured on the device to AWS IoT via MQTT.

SensorPublish is a container for the list of Sensor that are configured by customer. Each Sensor reads and publishes independently of other sensors.

SensorPublish notifies all Sensor instances in the list to stop and start.
