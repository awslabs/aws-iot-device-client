Detailed Description

Utility class intended to provide a somewhat 'unique' token.

We do not make any promises about the uniqueness of the generated token, only that it is hopefully unique enough for our purposes. IE we currently use the getRandomToken function to generate a token that can be used to map UpdateJobExecution requests back to their responses (in which only a few should be in flight at any given time) but this function would not be ideal for keys across a large store of data.
