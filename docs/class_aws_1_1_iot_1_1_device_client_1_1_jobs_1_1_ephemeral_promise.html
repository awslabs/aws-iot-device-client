Detailed Description

template<typename T>
class Aws::Iot::DeviceClient::Jobs::EphemeralPromise< T >

Provides the ability to specify a point of expiration for a promise.

This class allows you to specify a point of expiration for a promise indicating that the promise's time of usefulness is expired. This is used in our std::map of UpdateJobExecution promises, in case we fail to erase a promise due to an exception or interruption of some kind. This way, we don't leak the promises.
